# Personal Portfolio

A modern and responsive portfolio website built with **React** and **Vite** to showcase my projects, skills, and experience as a Front-End Developer.

## Features

- Responsive design for all devices
- Smooth animations with Framer Motion
- Project showcase with live demos
- Skills & technologies section
- Contact form powered by EmailJS
- Modern UI with clean and minimal design

## Tech Stack

- React 19
- Vite
- JavaScript (ES6+)
- Sass (SCSS)
- Framer Motion
- EmailJS
- React Icons
